# Mnemo v0.4.0-rc1 — `mnemo-graph` crate + Letta shared memory + golden fixture

Released 2026-04-25. Tag: [`v0.4.0-rc1`](https://github.com/sattyamjjain/mnemo/releases/tag/v0.4.0-rc1). PR: [#50](https://github.com/sattyamjjain/mnemo/pull/50). Merge commit: `94519f8`.

Release candidate stacking on top of v0.3.4. Lands three of the four follow-on tasks from the 2026-04-25 prompt; Mem0g graph-extraction toggle waits for v0.4.0 final because it depends on the LLM extractor leaving stub state.

## Highlights

- **`mnemo-graph` crate.** New workspace member for Graphiti-inspired bitemporal graph state. `TemporalEdge { src, dst, relation, valid_from, valid_to, confidence, recorded_at }` model. `DuckGraphStore` impl. `graph_expand(seed, depth, as_of)` BFS that respects fact validity. The headline integration test asserts the supersession property: an `as_of` query *between* a fact and its supersession returns the original answer; an `as_of` query *after* returns the new one. Concept doc: [`docs/src/concepts/temporal-edges.md`](../docs/src/concepts/temporal-edges.md).
- **`MnemoLettaShared` adapter.** Letta Conversations-style shared-memory shape — `attach`/`detach`/`list_participants`/`read`/`write` over Mnemo memories tagged `conversation:<id>` and `participant:<agent_id>`. Cross-participant writes within 60s surface via `overlapping_writes_within()` for operator inspection; conflict resolution at recall time via Mnemo's evidence-weighted scoring. Doc: [`docs/src/integrations/letta-conversations.md`](../docs/src/integrations/letta-conversations.md).
- **First golden DuckDB fixture.** `crates/mnemo-core/tests/golden/v0_3_4.mnemo.db` committed; `migration_roundtrip.rs` opens it and asserts the migrator stamps `CURRENT_PERSISTENCE_VERSION = 4` and round-trips exactly 5 records. The v0.1.1 / v0.3.0 historical fixtures remain blocked because those git tags don't exist on this repo — see [#38 comment](https://github.com/sattyamjjain/mnemo/issues/38#issuecomment-4319897458) for the gap analysis.
